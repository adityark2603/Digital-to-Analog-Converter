# Digital-to-Analog-Converter
A 4-bit R-2R Digital-to-Analog Converter (DAC) block implemented using Cadence Virtuoso tool. The DAC converts 4-bit digital inputs (D0-D3) into a proportional analog voltage output

### <ins>Common Source Amplifier</ins>
Implemented using a single NMOS-PMOS pair. Acts as a voltage amplifier stage to improve the drive capability of the DAC output.
